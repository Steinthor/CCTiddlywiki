Skinny Tiddlers ccTiddly Plugin

Status : Experimental

Designed for large tiddlywiki files. By only loading up the meta data for each tiddler the thoery is that we can speed up loading times for large tiddlywiki files.

Unfortunatly the use of in_array is proving slow. This plugin requires some more work before being used properly.
